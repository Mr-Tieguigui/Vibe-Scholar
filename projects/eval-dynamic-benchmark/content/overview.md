# Dynamic Benchmark Generation to Prevent Overfitting

## Definition

**Definition:** Implement a benchmark generation system that creates novel test instances on-the-fly, preventing models from memorizing static test sets.

## Motivation

This project addresses a gap in Evaluation & Benchmarking research. Implement a benchmark generation system that creates novel test instances on-the-fly, preventing models from memorizing static test sets. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
