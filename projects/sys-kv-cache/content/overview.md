# KV-Cache Compression for Long-Context Serving

## Definition

**Definition:** Benchmark KV-cache compression methods (quantization, eviction, streaming) for serving long-context LLMs under memory constraints.

## Motivation

This project addresses a gap in Systems & Tooling research. Benchmark KV-cache compression methods (quantization, eviction, streaming) for serving long-context LLMs under memory constraints. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
