# Efficient Vision Transformer Pruning for Edge Deployment

## Definition

**Definition:** Develop structured pruning strategies for ViT models targeting 10x FLOP reduction while maintaining 95% ImageNet accuracy.

## Motivation

This project addresses a gap in Computer Vision research. Develop structured pruning strategies for ViT models targeting 10x FLOP reduction while maintaining 95% ImageNet accuracy. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
