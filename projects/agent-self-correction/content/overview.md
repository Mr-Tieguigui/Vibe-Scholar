# Self-Correction Mechanisms in Code-Writing Agents

## Definition

**Definition:** Evaluate when and how code-writing agents can self-correct errors, measuring correction success rate and compute overhead.

## Motivation

This project addresses a gap in Agentic Systems research. Evaluate when and how code-writing agents can self-correct errors, measuring correction success rate and compute overhead. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
