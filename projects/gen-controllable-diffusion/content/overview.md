# Controllable Diffusion with Compositional Prompts

## Definition

**Definition:** Evaluate and improve spatial/attribute control in text-to-image diffusion models using compositional prompt benchmarks.

## Motivation

This project addresses a gap in Generative Models research. Evaluate and improve spatial/attribute control in text-to-image diffusion models using compositional prompt benchmarks. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
