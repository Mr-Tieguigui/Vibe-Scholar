# Foundation Models for Single-Cell RNA-seq Analysis

## Definition

**Definition:** Evaluate foundation models (scGPT, Geneformer) for cell type annotation, gene perturbation prediction, and batch correction tasks.

## Motivation

This project addresses a gap in Computational Biology research. Evaluate foundation models (scGPT, Geneformer) for cell type annotation, gene perturbation prediction, and batch correction tasks. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
