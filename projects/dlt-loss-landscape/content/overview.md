# Loss Landscape Topology of Fine-Tuned LLMs

## Definition

**Definition:** Analyze how fine-tuning alters the loss landscape topology of pre-trained LLMs using Hessian spectral analysis and mode connectivity.

## Motivation

This project addresses a gap in Deep Learning Theory research. Analyze how fine-tuning alters the loss landscape topology of pre-trained LLMs using Hessian spectral analysis and mode connectivity. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
