# Out-of-Distribution Detection for Medical Imaging

## Definition

**Definition:** Build and evaluate OOD detection methods for chest X-ray classifiers to flag distribution-shifted inputs before clinical deployment.

## Motivation

This project addresses a gap in Computer Vision research. Build and evaluate OOD detection methods for chest X-ray classifiers to flag distribution-shifted inputs before clinical deployment. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
