# Protein Function Prediction from Structure Using GNNs

## Definition

**Definition:** Benchmark graph neural networks for predicting protein function from AlphaFold-predicted structures, comparing against sequence-only baselines.

## Motivation

This project addresses a gap in Computational Biology research. Benchmark graph neural networks for predicting protein function from AlphaFold-predicted structures, comparing against sequence-only baselines. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
