# Long-Term Memory Architectures for Persistent Agents

## Definition

**Definition:** Compare memory architectures (vector DB, structured KG, episodic buffer) for agents operating across multi-day task horizons.

## Motivation

This project addresses a gap in Agentic Systems research. Compare memory architectures (vector DB, structured KG, episodic buffer) for agents operating across multi-day task horizons. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
