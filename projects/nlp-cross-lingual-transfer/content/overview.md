# Cross-Lingual Zero-Shot Transfer for Low-Resource NER

## Definition

**Definition:** Evaluate cross-lingual transfer from high-resource to low-resource languages for named entity recognition using multilingual transformers.

## Motivation

This project addresses a gap in NLP & Language Models research. Evaluate cross-lingual transfer from high-resource to low-resource languages for named entity recognition using multilingual transformers. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
