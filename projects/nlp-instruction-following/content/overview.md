# Instruction-Following Fidelity Under Constraint Stacking

## Definition

**Definition:** Measure how LLM instruction-following accuracy degrades as the number of simultaneous constraints in a prompt increases.

## Motivation

This project addresses a gap in NLP & Language Models research. Measure how LLM instruction-following accuracy degrades as the number of simultaneous constraints in a prompt increases. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
