# User Preferences for AI Explanation Modalities

## Definition

**Definition:** Study user preferences across explanation types (natural language, feature attribution, counterfactual, example-based) in high-stakes decision support.

## Motivation

This project addresses a gap in Human-Computer Interaction research. Study user preferences across explanation types (natural language, feature attribution, counterfactual, example-based) in high-stakes decision support. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
