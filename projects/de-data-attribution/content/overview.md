# Training Data Attribution for Model Predictions

## Definition

**Definition:** Implement influence function and datamodel-based attribution methods to trace model predictions back to specific training examples.

## Motivation

This project addresses a gap in Data Engineering research. Implement influence function and datamodel-based attribution methods to trace model predictions back to specific training examples. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
