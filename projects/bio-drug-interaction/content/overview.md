# Drug-Drug Interaction Prediction with LLM Augmentation

## Definition

**Definition:** Augment graph-based DDI prediction with LLM-generated molecular descriptions and pharmacological knowledge, measuring interaction prediction improvement.

## Motivation

This project addresses a gap in Computational Biology research. Augment graph-based DDI prediction with LLM-generated molecular descriptions and pharmacological knowledge, measuring interaction prediction improvement. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions
