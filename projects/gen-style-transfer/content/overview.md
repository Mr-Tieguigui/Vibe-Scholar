# Zero-Shot Style Transfer via Diffusion Inversion

## Definition

**Definition:** Implement and benchmark zero-shot artistic style transfer using DDIM inversion and cross-attention manipulation techniques.

## Motivation

This project addresses a gap in Generative Models research. Implement and benchmark zero-shot artistic style transfer using DDIM inversion and cross-attention manipulation techniques. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
