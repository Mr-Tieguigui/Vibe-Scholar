# Optimal Data Mixing Ratios for Pre-Training

## Definition

**Definition:** Systematically search data mixing ratios across code, web text, books, and academic papers for pre-training, measuring downstream task impact.

## Motivation

This project addresses a gap in Data Engineering research. Systematically search data mixing ratios across code, web text, books, and academic papers for pre-training, measuring downstream task impact. Current approaches lack systematic evaluation under realistic constraints.
