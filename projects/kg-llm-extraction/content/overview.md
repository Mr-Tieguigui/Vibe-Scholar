# LLM-Based Knowledge Graph Construction from Unstructured Text

## Definition

**Definition:** Evaluate LLMs as knowledge graph constructors, measuring entity/relation extraction quality, hallucination rates, and schema adherence.

## Motivation

This project addresses a gap in Knowledge Graphs research. Evaluate LLMs as knowledge graph constructors, measuring entity/relation extraction quality, hallucination rates, and schema adherence. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
