## Definition

**Definition:** Build automated detectors for reward model exploitation in RLHF, with a taxonomy of hacking strategies and mitigation benchmarks.

## Motivation

This project addresses a gap in Reinforcement Learning research. Build automated detectors for reward model exploitation in RLHF, with a taxonomy of hacking strategies and mitigation benchmarks. Current approaches lack systematic evaluation under realistic constraints.
