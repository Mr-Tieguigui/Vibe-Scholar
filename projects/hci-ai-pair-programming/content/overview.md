# AI Pair Programming Effectiveness Across Experience Levels

## Definition

**Definition:** Controlled study measuring how AI code assistants affect productivity, code quality, and learning outcomes for junior vs senior developers.

## Motivation

This project addresses a gap in Human-Computer Interaction research. Controlled study measuring how AI code assistants affect productivity, code quality, and learning outcomes for junior vs senior developers. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
