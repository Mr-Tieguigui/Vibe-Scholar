# Benchmark Contamination Detection Suite

## Definition

**Definition:** Build a suite of statistical tests for detecting training data contamination in LLM benchmark results, with synthetic control experiments.

## Motivation

This project addresses a gap in Evaluation & Benchmarking research. Build a suite of statistical tests for detecting training data contamination in LLM benchmark results, with synthetic control experiments. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
