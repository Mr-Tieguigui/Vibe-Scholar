# Prompt Sensitivity Analysis Across LLM Families

## Definition

**Definition:** Systematically measure how small prompt variations affect output quality across GPT-4, Claude, Gemini, and Llama families.

## Motivation

This project addresses a gap in NLP & Language Models research. Systematically measure how small prompt variations affect output quality across GPT-4, Claude, Gemini, and Llama families. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
