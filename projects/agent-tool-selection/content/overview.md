# Adaptive Tool Selection Under Uncertainty

## Definition

**Definition:** Develop agents that adaptively select tools based on uncertainty estimates, comparing UCB, Thompson sampling, and learned selection policies.

## Motivation

This project addresses a gap in Agentic Systems research. Develop agents that adaptively select tools based on uncertainty estimates, comparing UCB, Thompson sampling, and learned selection policies. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
