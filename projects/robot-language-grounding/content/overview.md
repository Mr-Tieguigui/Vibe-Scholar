# Language-Grounded Robot Task Planning

## Definition

**Definition:** Evaluate how well LLM-based planners translate natural language instructions into executable robot action sequences in household environments.

## Motivation

This project addresses a gap in Robotics & Embodied AI research. Evaluate how well LLM-based planners translate natural language instructions into executable robot action sequences in household environments. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
