# Real-Time Hallucination Detection in RAG Pipelines

## Definition

**Definition:** Build and evaluate hallucination detectors that operate in real-time within RAG pipelines, flagging unsupported claims before user delivery.

## Motivation

This project addresses a gap in Safety & Alignment research. Build and evaluate hallucination detectors that operate in real-time within RAG pipelines, flagging unsupported claims before user delivery. Current approaches lack systematic evaluation under realistic constraints.

## Research Questions

- **Q1:** How does the proposed approach compare to existing baselines?
- **Q2:** What are the key factors driving performance differences?
- **Q3:** How sensitive is the method to hyperparameter choices?
